# GIG-JS-TEST
Address book using JavaScript, CSS, HTML and Angular


### Installation


Clone repository:

```sh
$ git clone https://github.com/camiaguado/GIG-JS-TEST.git
$ cd GIG-JS-TEST
```



Install the dependencies and start the server.

```sh
$ npm install
$ node server.js
```

Open the app:

[Localhost](http://localhost:3002)
