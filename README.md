# GIG-JS-TEST
Single Page Application (SPA) address book using JavaScript, HTML and CSS.

### Installation


Clone repository:

```sh
$ git clone https://github.com/camiaguado/GIG-JS-TEST.git
$ cd GIG-JS-TEST
```



Install the dependencies and start the server.

```sh
$ npm install
$ node server.js
```

Open the app:

[Localhost](http://localhost:3002)
